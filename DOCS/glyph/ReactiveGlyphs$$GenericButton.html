org.sufrin.glyph.ReactiveGlyphs

GenericButton

abstract class GenericButton extends ReactiveGlyph

A GenericButton keeps track of whether the cursor is hovering over it, and whether it has been pressed but not released. If the mouse is pressed and then released it invokes react(). If the mouse is pressed over it, but released elsewhere, then there is no reaction.
