org.sufrin.glyph

Interaction

class Interaction extends AnyRef

When started, an Interaction hosts the given GUI root glyph in the given window. It uses a EventHandler to determine reactions to user- or system-generated events that take place in the window.

Various (overrideable) defaults define the size and position of the window, its title, its initial position on the screen, etc.
