org.sufrin.glyph.NaturalSize

RowGenerators

trait RowGenerators extends AnyRef

Provides various generators for a row of glyphs, with baseLine equivalent to the largest of the baseLines. The width is the sum of the glyph widths, and the height is the maximum glyph height.

RowGenerators with a specified foreground and background can be used in a Row expression; for example:

Row(fg=..., bg=...).atTop(glyph1, glyph2, ...)
Row(fg=..., bg=...).centered(glyph1, glyph2, ...)
