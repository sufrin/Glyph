  1.  new Dialogue(guiRoot: Glyph, location: RelativeTo = null, closeGlyph: Option[Glyph] = None, isModal: Boolean = true, isMenu: Boolean = false)

    guiRoot

    the glyph to be GUI for the dialogue when it is started

    location

    the location at which the top-left corner of the glyph is to be shown -- typically set non-null post-construction by (eg) North(glyph)

    closeGlyph

    If the popup is to have a close button showing g in its top left corner, then Some(g)

    isModal

    When true (the default case) this means that the popup grabs all input from its host window from the moment it is started until (just after) it is closed.

    isMenu

    When true this dialogue represents a popup menu
