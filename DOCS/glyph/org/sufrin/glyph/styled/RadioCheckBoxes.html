Companion object RadioCheckBoxes

class RadioCheckBoxes extends AnyRef

A group of captioned checkboxes, labelled with captions, and with prefer as the caption of the to-be-checkedinitially box. At most one box can be checked at a time: so whenever one is checked all the others are unchecked. When the currently-checked box is clicked and becomes unchecked, then the box (if any) captioned prefer is checked.
